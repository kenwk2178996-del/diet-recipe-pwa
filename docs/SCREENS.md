# 画面一覧

| 画面 | パス | 内容 |
|---|---|---|
| ログイン | /login | ログイン/新規登録/パスワード再設定/Googleログイン |
| ホーム | / | レシピ追加ボタン、最近保存、お気に入り |
| レシピ追加 | /add | URL/テキスト/スクショ/手動 の4タブ |
| 解析中 | /add (analyzing) | スケルトン + 進捗表示（フリーズ回避） |
| 確認・編集 | /add (confirm) | AI結果を編集して保存（材料/手順の追加・並び替え・タグ） |
| 詳細 | /recipes/[id] | 写真・PFC/kcal固定表示・材料(人数換算)・手順・メモ・作った記録・お気に入り・星 |
| 編集 | /recipes/[id]/edit | 既存レシピ編集（RecipeForm 共通化） |
| 検索 | /search | キーワード + タグ + 数値条件の複合検索、グリッド/リスト切替 |
| 設定 | /settings | アカウント、今月のAI解析使用量、ログアウト |

共通: 下部ナビ（ホーム/検索/追加/設定）、iOS向けクリップボードURL検出バナー。
